Chess Game

A simple interactive chess game built with React and styled with CSS. This project focuses on rendering a full chessboard, handling piece movement, and managing game state using React components. It was created as part of my learning journey to strengthen my understanding of component‑based UI design and state management.
Features

    Fully rendered chessboard using React components

    Click‑to‑move piece interactions

    Basic move validation

    Visual highlighting for selected pieces

    Responsive layout styled with CSS

Tech Stack

    React

    JavaScript

    CSS

Status

This project was built as part of my training and is being improved as I continue learning.
